# AI Object Detection with Voice Assistance

## Overview

This project is a real-time AI Object Detection system built using **Python**, **OpenCV**, and **Ultralytics YOLO-World**.

The application detects objects through a webcam, displays bounding boxes with object names, and announces the detected objects using text-to-speech.

It is designed as a beginner-friendly computer vision project to demonstrate real-time object detection and voice feedback.

---

## Features

- Real-time webcam object detection
- Voice announcements for detected objects
- Displays object names and confidence scores
- Detects multiple objects in a single frame
- Easy to customize by adding new object classes

---

## Technologies Used

- Python
- OpenCV
- Ultralytics YOLO-World
- PyTorch
- pyttsx3
